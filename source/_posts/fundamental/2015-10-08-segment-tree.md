# Overview

__[Segment tree](https://en.wikipedia.org/wiki/Segment_tree)__ is a tree data structure for storing intervals, or segments. 

Can be used to search __the max/min or sum values in a range__. 

1. modify = O(log n)

1. query = O(log n)

1. build = O(n)
